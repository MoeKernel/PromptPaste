## 当前技术栈

- .NET 8 / WPF
- SQLite：`Microsoft.Data.Sqlite`
- 托盘图标：`Hardcodet.NotifyIcon.Wpf`
- 图标资源：`resources/icons/app_icon.svg` 与 `resources/icons/app_icon.ico`
- 默认数据目录：`%USERPROFILE%\.promptpaste`
- 主要系统集成：剪贴板监听、全局热键、低层键盘钩子兜底、UI Automation/Win32 光标定位

## 推荐阅读顺序

1. 新开发者先读：当前功能说明 → 架构与代码结构 → 数据库与数据迁移。
2. 修改构建发布先读：开发环境与构建发布 → 发布验收清单。
3. 修改导入导出先读：JSON导入导出格式 → 数据库与数据迁移。
4. 修改快速候选或热键先读：当前功能说明 → 架构与代码结构 → 交互流程与测试验收。
