# 发布验收清单

## 自动化测试

- [ ] `dotnet test PromptPaste.sln -c Release` 通过。
- [ ] 测试覆盖文本处理、数据库 CRUD、分类/标签、回收站、分类合并和 JSON 导入导出。
- [ ] 已知限制：WPF 视觉与系统级热键/剪贴板行为仍以手工验收为准。

## 构建

- [ ] 运行 `pwsh.exe -ExecutionPolicy Bypass -File .\build.ps1` 成功。
- [ ] 根目录 `bin` 存在 `PromptPaste.exe`、`PromptPaste.dll`、`PromptPaste.deps.json`、`PromptPaste.runtimeconfig.json`。
- [ ] 发布包不包含 `.pdb`、`.xml`、`obj`、`.codegraph`、`src`、`main.py`、`requirements.txt`。
- [ ] `D:\Protable\PromptPaste` 已创建并同步最新发布文件；若文件被占用，已关闭正在运行的 PromptPaste 后重新部署。

## 启动与稳定性

- [ ] 双击 `bin\PromptPaste.exe` 能显示主窗口。
- [ ] 托盘图标显示正常，托盘菜单“显示/隐藏”“退出”有效。
- [ ] 关闭窗口按当前选项设置最小化到托盘或退出。
- [ ] 异常会写入 `%USERPROFILE%\.promptpaste\logs`。

## 数据安全

- [ ] 打开旧数据库时能自动迁移并保留数据。
- [ ] 导入 JSON、删除分类、移入/清空回收站、永久删除前会在 `%USERPROFILE%\.promptpaste\backups` 生成备份。
- [ ] 文件菜单可打开/切换数据库，最近数据库列表正常。
- [ ] 选项窗口可查看数据库路径、配置路径，并能手动备份。

## 核心功能

- [ ] 新建、编辑、复制、删除片段均有效。
- [ ] 回收站可恢复、永久删除、清空。
- [ ] 分类支持多级、拖拽移动、同名合并，不崩溃。
- [ ] 标签筛选支持输入匹配和多标签 chip。
- [ ] 带标签片段的卡片显示圆角标签 chip，编辑标签后卡片立即刷新。
- [ ] 搜索框可清空，空状态文案符合当前筛选状态。
- [ ] 剪贴板监听不会提示本软件内部复制出去的内容。

## UI 验收

- [ ] 顶栏两个输入框文字不被异常截断，并显示灰色提示文字。
- [ ] 卡片高度一致，标签 chip、日期、按钮位置稳定，无标签片段不出现空白标签占位。
- [ ] 热键唤出后无虚线焦点轮廓。
- [ ] Toast 提示不遮挡主要操作，能自动消失。
