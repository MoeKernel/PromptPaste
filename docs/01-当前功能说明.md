# 当前功能说明

PromptPaste 当前版本是 C# WPF 桌面应用，用于管理常用文本、AI 提示词和可变量替换的文本模板。

## 核心功能

- 片段管理：新建、编辑、复制、粘贴、删除、恢复和永久删除文本片段。
- 卡片视图：主区域以固定高度卡片展示标题、内容预览、分类、标签、使用次数和日期。
- 变量替换：内容中使用 `{{变量名}}`，复制或粘贴时可通过顶栏变量框或变量对话框替换。
- 搜索：顶栏搜索框按标题和内容过滤，支持一键清空。
- 剪贴板监听：检测外部复制的新文本，并在主界面顶部提示“保存为片段”。软件内部复制会被过滤。
- 全局热键：默认 `Ctrl+Shift+M` 唤出/隐藏窗口，可在选项中修改。
- 托盘运行：支持关闭到托盘、启动后最小化到托盘。
- 虚拟桌面：热键唤出时会尝试将窗口移动到当前 Windows 虚拟桌面。
- Toast 提示：复制、粘贴、导入、导出、删除、恢复等操作显示短提示。

## 分类功能

- 支持 Zotero 风格多级分类树。
- 片段可属于多个分类。
- 分类可拖拽移动到其他分类下。
- 同一父分类下分类名唯一；拖拽造成同名冲突时按合并处理。
- “包含子分类”放在菜单 `视图 -> 包含子分类` 中。
- 左侧“全部”按钮用于取消分类筛选。
- 新增分类按钮按当前激活分类决定创建位置：
  - 当前为“全部”：创建根分类。
  - 当前选中某分类：创建该分类的子分类。

## 标签功能

- 每个片段可拥有多个标签，卡片上会以圆角 chip 形式显示已有标签。
- 左侧标签筛选使用输入框 + 圆角 chip 的方式。
- 输入标签关键字后展示匹配建议；回车添加第一个建议。
- 多个标签筛选时，片段需同时包含所有已选标签。

## 回收站机制

- 普通删除会将片段移入回收站，不会立即物理删除。
- 回收站中支持恢复、永久删除和清空。
- 若一个片段存在于多个分类，右键可选择“从当前分类移除”。该操作只删除当前分类关系，不进入回收站。
- 只有真正移入回收站或永久删除时才影响片段本体。

## 多数据库功能

- 默认数据库位于 `%USERPROFILE%\.promptpaste\promptpaste.db`。
- 菜单 `文件 -> 打开数据库...` 可打开或创建新的 `.db` 数据库。
- 最近数据库路径保存在设置文件中，菜单可快速切换。
- 当前设计是同一窗口内热切换数据库，不打开独立窗口。

## 导入导出

- 菜单 `文件 -> 导入数据` / `导出数据` 使用 JSON 文件。
- 导出仅包含未删除片段。
- 导入前会提示确认并自动备份当前数据库。
- 帮助菜单提供 JSON 格式说明并支持复制示例。

## 选项与诊断

`文件 -> 选项...` 当前包含：

- 是否启用全局热键与热键录入。
- 是否启用剪贴板监听。
- 是否启动后最小化到托盘。
- 是否关闭窗口时最小化到托盘。
- 当前数据库路径、配置文件路径。
- 打开数据目录、打开日志目录、立即备份数据库。
