# PromptPaste

PromptPaste 是一款面向 AI 提示词、常用话术和文本模板的 Windows 桌面工具。你可以把常用文本保存成片段，通过分类、标签和搜索快速找到，并用快捷键在任意输入框附近唤出候选窗口进行插入。

## 主要功能

- **片段管理**：新建、编辑、复制、粘贴、删除、恢复和永久删除文本片段。
- **分类与标签**：支持多级分类、多个分类归属、标签 chip 和多标签筛选。
- **变量模板**：在内容中使用 `{{变量名}}`，复制或粘贴时填写变量值后自动替换。
- **快速候选**：在文本输入框中按快捷键，像输入法候选一样搜索并插入片段。
- **剪贴板监听**：检测外部复制的新文本，可一键保存为片段；软件内部复制会自动过滤。
- **导入导出**：使用 JSON 文件导入/导出片段数据。
- **托盘运行**：可关闭到托盘、启动后最小化到托盘。
- **日志查看**：帮助菜单内置日志查看器，方便排查热键、弹窗和运行问题。

## 快捷键

| 快捷键 | 作用 | 说明 |
|--------|------|------|
| `Ctrl+Shift+M` | 显示/隐藏主窗口 | 全局热键，可在 `文件 -> 选项...` 修改 |
| `Ctrl+Alt+Space` | 打开快速候选弹窗 | 全局热键，可在 `文件 -> 选项...` 修改 |
| `↑` / `↓` | 切换快速候选项 | 在快速候选弹窗中使用 |
| `Enter` | 插入当前候选 | 在快速候选弹窗中使用 |
| `Esc` | 关闭快速候选弹窗 | 在快速候选弹窗中使用 |

> 如果快捷键被其他软件占用，可以在 `文件 -> 选项...` 中重新录入组合键。快速候选热键和主窗口唤出热键不能相同。

## 快速候选怎么用

1. 把光标放到任意文本输入框中。
2. 按 `Ctrl+Alt+Space` 打开快速候选。
3. 输入关键词搜索标题、内容或标签。
4. 使用 `↑` / `↓` 选择候选。
5. 按 `Enter` 插入，或双击候选插入。

快速候选窗口会尽量靠近输入光标弹出；遇到屏幕边缘会自动避让。长标题、长内容和长标签可以鼠标悬停查看完整内容。

## 变量模板

片段内容中可以写变量：

```text
请帮我把以下{{文本类型}}改写成{{风格}}风格：
{{原文}}
```

复制或粘贴时：

- 如果只有一个变量，并且主界面顶部变量框有值，会直接使用该值替换。
- 如果有多个变量，或变量框为空，会弹出变量输入窗口。
- 填写完成后，PromptPaste 会把变量替换为实际内容再复制或粘贴。

## 常用操作

| 操作 | 方法 |
|------|------|
| 新建片段 | 点击右上角“新建”按钮 |
| 编辑片段 | 点击卡片上的“编辑” |
| 复制片段 | 点击卡片上的“复制”或右键选择复制 |
| 粘贴片段 | 双击卡片，或在快速候选中按 `Enter` |
| 删除片段 | 卡片右键或编辑入口中删除，先进入回收站 |
| 恢复片段 | 进入回收站后点击恢复 |
| 导入/导出 | 菜单 `文件 -> 导入数据` / `导出数据` |
| 查看日志 | 菜单 `帮助 -> 查看日志` |
| 查看 JSON 格式 | 菜单 `帮助 -> JSON格式说明` |

## 数据和配置位置

默认数据保存在当前 Windows 用户目录下：

| 类型 | 位置 |
|------|------|
| 默认数据库 | `%USERPROFILE%\.promptpaste\promptpaste.db` |
| 设置文件 | `%USERPROFILE%\.promptpaste\settings.json` |
| 日志目录 | `%USERPROFILE%\.promptpaste\logs` |
| 备份目录 | `%USERPROFILE%\.promptpaste\backups` |

你也可以通过 `文件 -> 打开数据库...` 打开或创建其他 `.db` 数据库。

## 安装与运行

当前版本是 C# / WPF Windows 桌面应用。

### 使用发布包

1. 获取发布目录或压缩包。
2. 双击运行 `PromptPaste.exe`。
3. 首次启动会自动创建默认数据目录和数据库。

### 从源码构建

环境要求：

- Windows 10/11
- .NET SDK 8.x

构建：

```powershell
dotnet build PromptPaste.sln -c Release
```

运行测试：

```powershell
dotnet test PromptPaste.sln -c Release
```

发布构建：

```powershell
powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\build.ps1
```

## 导入导出格式

导入导出使用 JSON 数组。最小示例：

```json
[
  {
    "title": "AI 提示词示例",
    "content": "请帮我把以下{{文本类型}}改写成{{风格}}风格。",
    "categoryPaths": ["AI提示词/写作"],
    "tags": ["prompt", "写作"]
  }
]
```

更多字段和规则可在应用内打开：

```text
帮助 -> JSON格式说明
```

## 问题排查

- 快捷键无效：检查是否被其他软件占用，进入 `文件 -> 选项...` 更换热键。
- 快速候选位置异常：打开 `帮助 -> 查看日志`，复制日志排查定位来源和 DPI 信息。
- 剪贴板未提示保存：确认 `文件 -> 选项...` 中已启用剪贴板监听。
- 数据异常或误删：查看 `%USERPROFILE%\.promptpaste\backups` 下的自动备份。
