# PromptPaste

PromptPaste是一款功能强大的剪贴板管理工具，专为存储和管理AI提示词、常用话术而设计。

## 功能特点

- **智能存储**：自动监听剪贴板，可选择性保存复制的文本
- **分类管理**：支持将内容分类为"AI提示词"、"商务话术"等不同类型
- **快速搜索**：实时筛选功能，快速找到需要的内容
- **一键复制**：双击即可将内容复制到剪贴板
- **变量替换**：支持在文本中使用`{{变量}}`格式，复制时自动替换为指定内容
- **使用统计**：记录每条内容的使用次数和最后使用时间
- **数据导入导出**：支持批量导入导出数据

## 安装说明

### 环境要求
- Python 3.8+
- 依赖库：tkinter, ttkthemes, pyperclip, sqlite3

### 安装步骤
1. 克隆或下载本仓库
2. 安装依赖：`pip install -r requirements.txt`
3. 运行应用：`python main.py`

## 使用指南

### 主界面
- **卡片视图**：以卡片形式展示所有存储的内容
- **侧边栏**：快速筛选不同类型的内容
- **搜索框**：输入关键词实时筛选内容
- **变量输入框**：输入变量值，用于替换文本中的`{{变量}}`

### 操作方式
- **添加内容**：点击"添加"按钮或从剪贴板监听弹窗保存
- **复制内容**：双击卡片即可复制内容到剪贴板
- **编辑/删除**：右键点击卡片，选择相应操作
- **导入/导出**：通过菜单栏的"文件"选项进行操作
